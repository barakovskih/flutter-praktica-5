# Карточка товара

Учебный проект на Flutter — практическая работа №7 «Управление компоновкой».

Одноэкранное приложение, которое показывает карточку товара.

## Что на карточке

- изображение товара с бейджем «Новинка»;
- заголовок;
- рейтинг и количество отзывов;
- описание;
- цена со скидкой: новая и зачёркнутая старая;
- кнопка действия.

## Виджеты

Карточка собрана из виджетов компоновки `Container`, `Column`, `Row`, `Text` и `Image`; бейдж поверх фото сделан через `Stack`, действие — кнопкой. Внешний `Container` с рамкой (`Border.all`) задаёт границу карточки. Оформление — Material 3.

## Стек

- Flutter 3.44, Dart 3.12;
- Material 3.

## Как запустить

```bash
flutter pub get
flutter run -d chrome
```

Вместо `chrome` можно выбрать Android-эмулятор или подключённое устройство.

## Структура проекта

- `lib/main.dart` — приложение и виджеты карточки;
- `assets/images/` — изображение товара.
